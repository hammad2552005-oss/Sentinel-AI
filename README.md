# Sentinel AI - Network Security Auditor
**Developed by: Hammad Khan** **Seat No: B2533037** **University: Benazir Bhutto Shaheed University Lyari (BS AI)**

## Project Overview
Sentinel AI is a professional network monitoring and security tool developed to detect active nodes within a local area network (LAN). It provides administrators with the capability to identify device hostnames, MAC addresses, and perform security mitigation through ARP termination.

## Core Features
* **Live Network Radar:** Real-time scanning of IP ranges.
* **Device Fingerprinting:** Identifying hostnames and hardware addresses.
* **Security Mitigation:** ARP Poisoning mechanism to disconnect unauthorized devices.
* **Professional GUI:** Dark-themed dashboard built with Python Tkinter.

## Technical Stack
* **Language:** Python 3.x
* **Core Library:** Scapy (Packet Manipulation)
* **Interface:** Tkinter
